MCP Client Configuration

Visual Studio Code

Reference: How to enable MCP support in VS Code

In VSCode:

  • Open Folder
  • Add .vscode/mcp.json
  • Paste this:
{
    "servers": {
        "joke-server": {
            "url": "https://mcp-joke-server.onrender.com/mcp/",
            //"url": "http://localhost:8000/mcp/", for localhost testing
            "headers": {
                "X-API-Key": "{{ api_key }}" // from JOKE_MCP_API_KEY env variable
            }
        }
    }
}

Testing Your Configuration

To test your setup, open the CHAT window in VS Code, select 'Agent' mode, and type something like 'tell a joke'.
